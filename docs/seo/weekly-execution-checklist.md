# Weekly SEO Execution Checklist (Aligned to Deploy Cadence)

This checklist assumes you ship website/content changes with npm run deploy.

## Cadence Model

- Monday: Plan and produce
- Tuesday: Publish and deploy
- Wednesday: Outreach wave 1
- Thursday: Follow-up and partnerships
- Friday: Measurement and backlog updates

## Weekly Checklist

### Monday: Build and Prep

- [ ] Select 1 primary linkable asset for this week
- [ ] Draft or update the asset page in src
- [ ] Prepare 3 outreach angles tied to current topics
- [ ] Add 10-15 new targets to target list
- [ ] Update outreach-tracker.csv rows for this week

### Tuesday: QA and Deploy

- [ ] Run local QA of changed pages
- [ ] Confirm title, meta description, and canonical are present on new/updated pages
- [ ] Confirm internal links to key assets are in place
- [ ] Deploy with npm run deploy
- [ ] Post-deploy smoke check:
  - [ ] Homepage loads
  - [ ] New/updated asset page loads
  - [ ] robots.txt loads
  - [ ] sitemap.xml loads

### Wednesday: Outreach Wave 1

- [ ] Send 8-12 personalized emails
- [ ] Log every email in outreach-tracker.csv
- [ ] Record response windows and follow-up dates
- [ ] Share published asset on LinkedIn and X

### Thursday: Follow-up and Co-Marketing

- [ ] Send follow-up to non-responders (5-7 days after first email)
- [ ] Pitch 2 co-marketing ideas to partner candidates
- [ ] Update tracker statuses (Contacted, Follow-up Sent, In Conversation)

### Friday: Review and Improve

- [ ] Count sent emails, replies, and positive responses
- [ ] Count links acquired and new referring domains
- [ ] Note best-performing pitch angle
- [ ] Decide next week asset and target categories
- [ ] Add improvements to backlog before Monday

## 6-Week Sprint Targets

- Week 1-2:
  - [ ] Publish 2 linkable assets
  - [ ] Send 20-25 outreach emails
- Week 3-4:
  - [ ] Publish 1 additional asset
  - [ ] Start partner/co-marketing collaborations
  - [ ] Secure first 3-5 editorial links
- Week 5-6:
  - [ ] Publish data insight post
  - [ ] Run micro PR outreach wave
  - [ ] Reach 8-15 new referring domains total

## Deploy Day Guardrails

Before every deploy that includes SEO pages:

- [ ] No placeholder links remain
- [ ] All new links are crawlable and absolute where needed
- [ ] Link targets open correctly
- [ ] Added page is linked from at least one existing page
- [ ] URLs intended for indexing are included in sitemap if needed
