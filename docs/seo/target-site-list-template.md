# Target Site List Template

Use this sheet to qualify and prioritize outreach targets.

## Qualification Rules

A site is a good target when:

- Topic overlap is strong (fitness, strength, coaching, AI tools, health tech)
- Content is active (published in last 6 months)
- Site has real editorial standards (not obvious link farm)
- It has a clear contributor or contact path
